# ComplyHealth

ComplyHealth is an easy-to-use chronic health management app built with Flutter. This project was developed as a startup venture with a co-worker. The current state of the repo reflects the handoff point when I transitioned off the project.

--- 

## Motivation

Development of ComplyHealth was driven by the desire to provide those living with chronic health issues a simple and convenient way to manage their conditions. As a nurse, I've seen many patients come into the hospital with no idea why they take their medications or what they take them for. ComplyHealth was built with these principles:
 - **Simplicity**: The target audience for this application is an aging population. Therefore, the design philosophy was: if you can use Facebook, you can use this app.
 - **Adherence**: The user should be motivated to stay adherent to their medications through easy medication tracking.
 - **Privacy**: Most health applications sell your data to make an income. ComplyHealth doesn't do that.

## Quick Start

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed
- A connected device or emulator

### Run these 4 steps
``` bash
# 1. Clone the repo
git clone https://github.com/pndaRN/ComplyHealth && cd ComplyHealth
```
``` bash
# 2. Switch to working branch
git switch develop # Latest App Release

# or 

git switch localtest # Latest Testing Release
```
``` bash
# 3. Install dependencies
flutter pub get
```
``` bash
# 4. Run the app
flutter run
```
